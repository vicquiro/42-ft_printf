# 42-ft_printf


## Requirements

The functions are written in __C language__ and need the `gcc` compiler, with `<unistd.h>`, `<stdlib.h>` and `<stdarg.h>` standard libraries to run.

## Instructions

**Compiling the library**

```shell
$ make
```

**Using it in your code**

```C
#include "ft_printf.h"
```
